# Test Fixtures

This directory contains test fixtures for scenario tests.

## Repository Fixtures

Pre-built test repositories with common patterns:
- `simple-pr/` - One file, one comment
- `multi-file-pr/` - Multiple files, multiple comments
- `conflict-scenario/` - Merge conflicts
- `large-pr/` - Many files, many comments

## PR Data Fixtures

JSON/YAML files with PR metadata:
- Comment sets (inline, general, threads)
- Bot response patterns
- State snapshots
